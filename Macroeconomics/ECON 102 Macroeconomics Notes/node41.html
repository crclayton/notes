Shifts in Equilibrium National Income

So 12#12 doesn't want to change, but it can and much effort is put into ensuring it does.

If the AE function shifts up, it intersects the 45 degree line further along the x-axis, increasing 12#12. The AE function shifts upwards whenever the C or the I functions increase.

Figure 5: Shifts in AE Function

The equilibrium point can also change if the AE function rotates. Recall the slope of the AE function is the MPS, and so as MPS increases, AE becomes steeper and 12#12 increases.
